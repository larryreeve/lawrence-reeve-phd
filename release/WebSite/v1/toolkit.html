Background


The BioTUS Summarization and Annotation Toolkit allows you to perform biomedical concept annotation and text summarization using biomedical concepts on a local machine. The toolkit requires the UMLS MetaMap application and its data resources, which must be downloaded separately.

Software

Summarization and Annotation Toolkit 1.0

Environment

Java 1.5 or greater (separate download)
UMLS MetaMap (separate download)
Apache Commons CLI library (included in toolkit)
Apache Commons Collections library (included in toolkit)

Configuration

The annotator is invoked using the sample Windows batch file called bioannotator.bat.
The summarizer is invoked using the sample Windows batch file called biosumm.bat.
bioannotator.bat and biosumm.bat must be modified to set three environment variables:

  UMLS_HOME - points to the location where UMLS exists

  DATA_HOME - points to the location where data for the annotator and summarizer is stored.
    The structure of the DATA_HOME area is pre-defined. A sample of the structure is included in the toolkit.

  RELEASE_HOME - points to the location where the toolkit exists

No additional configuration is required if you use MetaMap Transfer as the annotator.

To use CONANN as the annotator, the MetaMap Transfer data resources must first be pre-processed for use by CONANN.
CONANN uses the strict version of the MetaMap Transfer data, located in /UMLS_HOME/mmtx/nls/mmtx/data/2004.
Pre-processing is invoked by starting bioannotator.bat:   bioannotator -preprocess

Pre-processing the data can take several hours.

Annotation

A sample Windows batch file called bioannotator.bat is included in the toolkit.
Sample usage of bioannotator.bat is given in several examples below.
For a list of all possible parameters, execute bioannotator without any parameters.

Example command line to show all parameters:
  bioannotator

Example command line using MetaMap Transfer as the annotator:
  bioannotator -inputfile abstract.txt -outputfile output.txt -annotator mmtx -verbose

Example command line using CONANN as the annotator:
  bioannotator -inputfile abstract.txt -outputfile output.txt -annotator duist -verbose


Summarization

A sample Windows batch file called biosumm.bat is included in the toolkit.
Sample usage of biosumm.bat is given in several examples below.
For a list of all possible parameters, execute biosumm without any parameters.

Example command line to show all parameters:
  biosumm

Example command line using MetaMap Transfer as the annotator:
  biosumm -annotator mmtx -inputfilefull article.txt -outputtype summary -outputfile ArticleSummary_10pct.spl -outputoptions compressionrate=10;format=spl -scorer freqdist -scoringoptions useconcepts=false;similarityfunction=Dice

Example command line using CONANN as the annotator:
  biosumm -annotator duist -inputfilefull article.txt -outputtype summary -outputfile ArticleSummary_10pct.spl -outputoptions compressionrate=10;format=spl -scorer freqdist -scoringoptions useconcepts=false;similarityfunction=Dice
